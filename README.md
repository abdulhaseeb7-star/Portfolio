# Abdul Haseeb — Personal Portfolio

A clean, dark-themed personal portfolio website built with Vite, Tailwind CSS, and Vanilla JavaScript.

🌐 **Live Site:** [Coming Soon on Netlify]

---

## 🛠️ Tech Stack

- **Vite** — lightning-fast build tool
- **Tailwind CSS v3** — utility-first CSS framework
- **Vanilla JavaScript** — no frameworks
- **AOS** — scroll animations

---

## 📁 Project Structure
my-portfolio/
├── public/          → static assets (CV, images)
├── src/
│   ├── main.js      → imports CSS + initializes AOS
│   └── style.css    → Tailwind directives + global styles
├── index.html       → entire single-page website
├── tailwind.config.js
├── postcss.config.js
└── vite.config.js
## 🚀 Run Locally

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```
